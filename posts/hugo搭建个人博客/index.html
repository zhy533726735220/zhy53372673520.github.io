Hugo搭建个人博客

December 10, 2019

Hugo

Go语言实现的一个博客生成器 世界上最快的博客生成器

Js实现的博客生成器叫Hexo

安装Hugo

官方教程

  • Mac安装方式

    • brew install hugo
    • hugo version

  • windows安装方式

    • 去Hugo releases页面下载hugo_xxx_Windwos-64bit.zip
    • 解压，把hugo.exe放到D:\Software\hugo\hugo.exe
    • 把D:\Zwofteware\hugo加到PATH
    • 重启终端，运行hugo version查看版本

快速搭建博客

  • 看文档

    • 进入Hugo官网，点击Quick Start快速开始

    • 执行Step2开始，直到Step7结束

      • hugo new site quickstart
        • quickstart：github账号的名字(全小写) + github.io + creator
          title: "Test"
    date: 2019-12-10T16:52:43+08:00
    # true为草稿 false为发布
    draft: true

      • 得到一个public目录，这就是我们的博客站点

      • hugo server -D  可以预览草稿

      • hugo server 可以预览非草稿

      • hugo -D

    • 上传到Github

      • 在博客生成器中创建.gitgnore文件 - 忽略掉publicw文件夹

      • 在public文件夹中创建git仓库
